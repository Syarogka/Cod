# Orna Codex RU Scraper

Запуск на GitHub:
1. Загрузите файлы в новый репозиторий.
2. Откройте Actions.
3. Выберите **Download Russian Orna Codex**.
4. Нажмите **Run workflow**.
5. После завершения скачайте artifact `orna_codex_ru_database`.

Результат содержит:
- `orna_ru_database.json` — база русских карточек;
- `orna_ru_images.zip` — скачанные изображения;
- `report.txt` — отчёт.
