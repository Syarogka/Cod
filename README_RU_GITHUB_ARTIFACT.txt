# Скачать недостающие картинки Orna через GitHub Actions отдельным ZIP

Этот пакет **не меняет сайт** и **не делает commit**. GitHub Actions только скачивает недостающие картинки с `https://playorna.com/static/img/...` и выдаёт отдельный ZIP-артефакт.

В списке: 757 картинок.

## Как использовать

1. Распакуй этот архив.
2. Закинь в свой GitHub-репозиторий две папки:
   - `.github/workflows/`
   - `tools/missing_assets/`
3. Сделай commit и push.
4. Открой репозиторий на GitHub.
5. Перейди: **Actions → Download Orna missing images as ZIP → Run workflow**.
6. Дождись завершения.
7. Открой завершённый запуск workflow.
8. Внизу страницы будет раздел **Artifacts**.
9. Скачай артефакт **orna_missing_images_downloaded**.
10. Загрузи скачанный ZIP в ChatGPT.

## Что будет внутри ZIP-артефакта

- `static/img/...` — скачанные картинки в правильных папках;
- `downloaded_ok.txt` — список того, что скачалось;
- `downloaded_failed.txt` — список того, что не скачалось.

## Важно

- Сайт загружать в этот пакет не нужно.
- Workflow не будет ничего вставлять в сайт автоматически.
- После того как ты загрузишь мне ZIP-артефакт, я добавлю картинки в сайт и соберу новый архив.
